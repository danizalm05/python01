(Ctrl+Shift+P)  command Palette
(Ctrl+Shift+X)  open the Extensions view 

1. Open the Command Palette (Ctrl+Shift+P), start typing the Python: Create Environment command to search, and then select the command.

2. The command presents a list of environment types, Venv or Conda. select Venv.

3. Select the your interpreter.

4. After selecting the interpreter, a notification will show the progress of the environment creation and the environment folder (/.venv) will appear in your workspace.

5. Ensure your new environment is selected by using the Python: Select Interpreter command from the Command Palette.

               Configure and run the debugger

 1. open the Extensions view (Ctrl+Shift+X) and search for @installed python debugger. You should see the Python Debugger extension listed in the results.		
 2. Set a breakpoint   by   clicking  in the editor's left gutter, next to the line numbers., a red circle appears in the gutter. 
 3.  press F5,  to initialize the debugger.
 4. Select Python File, which is the configuration that runs the current file shown in the editor using the currently selected Python interpreter.
