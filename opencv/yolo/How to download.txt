                           OIDv6 Toolkit
						   --------------
pip install oidv6
Downloading  dataset of 100 photo of Alpaca
1. start  anconda promper 
2. move to  the location you want to store the data, for example:   C:\Users\ .....\Pictures 
3. run the next comman to dwonload  100 Alpaca photo
      oidv6 downloader --classes Alpaca --type_data train --limit 100

 output :
     in the directory where you run the 'oidv6' you will  get:
      OIDv6/
       └── train/
             └── Alpaca/
             ├── img1.jpg
             ├── img2.jpg
           └── ...
 
 CVAT: (Computer Vision Annotation Tool)  This is a labling tool 
 https://www.cvat.ai/
 1. log in
 2. From project Create a new project( the '+' sign on the right)
 3. Chose name alpaca_detector
 4. For labels add one label = 'alpaca'
 5. At the bottem left pree '+'  and chose create a new task.
 6. chose name 'detect001'
 7. Go to the upload section and chose the directory and from this directory chose 
    the files  you want to upload.
 8. Press submit and open.
 9. After the photos are uploaded , press the job number.
10. from the left side chose 'draw ractengle' . 
11. Chose the shape option and draw the rectangle. chose 'n' as a shortcut for chosing another
    bounding box. Once finsed save use Ctrl-s
12 Chpse the '>' to move to the next photo.	
13 Press 'task' from the main menu
14. Press action and choose export task database.  
15. From the new window in export format chose  YOLO 1.1
16. Press OK
17. In the main menu press request  right click on the task and download.
18.  create two directorys 'images' the will contain all the images    and 'labels' that will contain the labels files. 
     for each image file there is a label file.
